=====================
Mongo DB installation
=====================

This model manager service uses the mongoDB server community edition to store its documents and collections

Note: client who want to run this mS on their own Linux environments have to install mongoDB server

Installation detail available at mongodb wiki

  https://docs.mongodb.com/manual/administration/install-on-linux/
